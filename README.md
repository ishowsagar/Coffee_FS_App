# Coffee_FS_App

Coffee_FS_App is a full-stack coffee catalog experience built for clean API design and a fast modern UI.

It combines a Go backend and a React frontend in one focused workspace, so you can build, test, and ship both layers together without context switching.

## Introduction

From bean origin to roast details, this app is designed to showcase how a practical full-stack product should feel:

- Fast API responses powered by Go
- Simple, responsive React interface
- Clear separation between client and server
- One repository, one workflow, zero confusion

## Run

### Frontend

```bash
cd coffee-client
npm install
npm run dev
```

### Backend

```bash
cd coffee-server
go run cmd/server/main.go
```
